AI-powered CSP generator. Creates secure Content Security Policy headers tailored to your app.

Generate Content Security Policy headers for your site.

## Install

```bash
npm install -g ai-csp
```

## Usage

```bash
npx ai-csp
# Analyzes project and generates CSP headers

npx ai-csp ./my-app -o security.md
```

```bash
export OPENAI_API_KEY=sk-...
```
